## Swift

You can find the Swift wrapper plugin [here](https://github.com/csjones/hookflow-plugin).

Utilize hookflow in your Swift project using Swift Package Manager:

```swift
.package(url: "https://github.com/csjones/hookflow-plugin.git", exact: "1.0.0"),
```

Or, with [mint](https://github.com/yonaskolb/Mint):

```bash
mint run csjones/hookflow-plugin
```
