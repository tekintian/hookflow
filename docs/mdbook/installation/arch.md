## AUR for Arch

- Official [AUR package](https://aur.archlinux.org/packages/hookflow) (compiles from sources)
- Community [AUR package](https://aur.archlinux.org/packages/hookflow-bin) (delivers pre-compiled binaries)

```sh
# To compile from sources
yay -S hookflow

# To install only executable
yay -S hookflow-bin
```
