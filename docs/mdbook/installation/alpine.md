## APK packages for Alpine

```sh
sudo apk add --no-cache bash curl
curl -1sLf 'https://dl.cloudsmith.io/public/tekintian/hookflow/setup.alpine.sh' | sudo -E bash
sudo apk add hookflow
```

See all instructions: https://cloudsmith.io/~tekintian/repos/hookflow/setup/#formats-alpine
